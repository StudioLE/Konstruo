This is an experimental WebAssembly build of Konstruo.

There is a known issue with orbiting resulting in a blank screen on WebAssembly builds.

Download native binaries from GitHub Releases for the best graphics and performance.

Pan with middle mouse button or W, A, S, D keys.

Orbit with ⇧ + middle mouse button or ⇧ + W, ⇧ + A, ⇧ + S, ⇧ + D keys.

Zoom with scroll wheel or +, - keys.
